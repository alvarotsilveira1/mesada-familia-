# 💰 Mesada Família

App de mesada para família de 4 pessoas (pais e 2 filhos). Sistema completo de gestão de mesada por desempenho escolar, tarefas, iniciativas, controle de tempo de tela, módulo bancário, empreendedorismo e fechamento mensal.

## 🚀 Como usar

1. Acesse o app pelo link do GitHub Pages no Safari do iPhone
2. Toque em "Compartilhar" → "Adicionar à Tela de Início"
3. Cada membro da família faz login com seu PIN

## 📋 Funcionalidades

### 📚 Desempenho escolar
- Disciplinas A (peso maior) e B (peso menor)
- Avaliações APA, AT1, AT2 disponíveis o ano todo
- Lançamento de notas com foto da prova
- Ocorrências escolares com foto e desconto

### 🏠 Tarefas e iniciativas
- 9 tarefas configuráveis (passear cachorro, lavar louça, etc)
- Iniciativas espontâneas com tier de valor
- Foto/vídeo como evidência

### 📱 Tempo de Tela
- 12 categorias da Apple (Jogos, Redes, Educação, etc)
- Cada uma com tipo configurável: Premia / Penaliza / Neutra
- Lançamento em HH:MM (total da semana)
- Aprovação pelos pais

### 🏦 Banco
- Importação de extratos: Inter, NgCash, PayPal
- 14 categorias financeiras com sugestão automática
- Dashboard de entradas e saídas
- Regra de poupança 30% configurável

### 💼 Empreendedorismo
- Vendas online, físicas e prestação de serviços
- Mensagens motivacionais por marco
- Dashboard separado do total

### 📋 Fechamento Mensal
- Banner automático no início de cada mês
- Checklist obrigatório
- Apontamento de doação com feedback educativo
- Lock após fechamento (só pais editam)
- Relatório completo + WhatsApp PIX

## 🔄 Sistema de atualizações

O app verifica nova versão a cada 6 horas. Quando detecta, mostra banner "🆕 Nova versão disponível". Toque pra recarregar.

## 💾 Armazenamento

Por padrão, os dados ficam no iPhone (localStorage). Para sincronizar entre iPhones, configure o Firebase em Configurações → Sync.
